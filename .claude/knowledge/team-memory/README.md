# 강팀 브레인 (team-memory)

이 폴더는 강팀이 *여러 프로젝트를 거치면서 학습한 것*을 누적하는 자리.

전역 설치 시(`~/.claude/knowledge/team-memory/`로 복사됨) 모든 프로젝트의 강팀이 이걸 읽고 시작 — *원점에서 시작하지 않는다*.

## 구조

```
team-memory/
├── lessons/    프로젝트별 회고 시트 (templates/lesson.md 형식)
├── decisions/  팀 차원 의사결정 (ADR 형식, 한 결정 한 파일)
├── patterns/   여러 프로젝트에서 통한 패턴 (재사용 가능)
└── domain/     특정 영역에서 얻은 지식 (이커머스, 모임앱, 등)
```

## 추가 방법

1. 프로젝트 끝에 `templates/lesson.md` 복사해 회고 작성.
2. *글로벌로 옮길 항목*에 체크된 것만 → 이 `Ai_Team` 레포에 PR.
3. 머지되면 `install-global` 스크립트 재실행으로 모든 PC에 반영.

## 새 프로젝트 시작 시

강팀장이 *반드시* 이 폴더를 한 번 훑고 시작 — "이 프로젝트와 관련된 과거 결정·패턴이 있는가?" 확인.
