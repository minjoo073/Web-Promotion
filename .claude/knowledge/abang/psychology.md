# 심리 레버 → 수익 환금 카탈로그

아뱅이 아이디어를 낼 때 참조하는 심리 레버 사전. 각 레버는 **무엇을 자극하는가 → 어떻게 행동을 끌어내는가 → 수익으로 어떻게 환금하는가** 의 세 줄로 구성.

답변할 때는 *최소 1개 레버를 명시*하고, 가능하면 2~3개를 *조합*해서 효과를 증폭한다.

---

## 1. 경쟁 (Competition)

- **자극**: "남보다 앞서야 한다", "뒤처지기 싫다"
- **행동 유도**: 랭킹·리더보드·등급·점수 가시화 → 자발적 반복 사용
- **수익 환금**:
  - 상위권 진입을 위한 **유료 부스트**(아이템·광고권·시간 단축권)
  - 등급에 따라 노출 우선순위 차등 → 판매자가 등급 유지 위해 활동량↑ → 수수료↑
  - 시즌제 리셋으로 *재경쟁* 유도 → 매 시즌마다 수익 사이클

**조합 추천**: 사회적 증거(#5), FOMO(#4), 매몰비용(#9)

---

## 2. 인정/관심 욕구 (Validation, Attention)

- **자극**: "내 존재를 알아봐줘", "도파민"
- **행동 유도**: 좋아요·댓글·구독·공유·뱃지 → 콘텐츠 생산 반복
- **수익 환금**:
  - 콘텐츠 생산자가 늘면 **콘텐츠 공급 비용 0** + 광고 인벤토리↑
  - 인기 생산자에게 **수익 배분** → 더 큰 생산자가 자기 팬을 끌고 옴 (유저→유저 유입)
  - "더 많은 관심"을 위한 **유료 노출/꾸미기 상품**

**원본 케이스**: 쇼핑몰 코디 마켓의 자동 응원 댓글 (인정 욕구로 제작자 락인)
**조합 추천**: 자기효능감(#8), 경쟁(#1)

---

## 3. 손실 회피 (Loss Aversion)

- **자극**: "얻는 기쁨"보다 "잃는 고통"이 약 2배 강하다 (카너먼)
- **행동 유도**: "지금 안 하면 못 한다", "포기하면 X가 사라진다" → 즉시 결제·즉시 행동
- **수익 환금**:
  - **무료 체험 → 결제 전환** (체험 종료 = 잃는 것)
  - 적립금 **유효기간** 설정 → 소멸 직전 재방문·재구매
  - **연속 출석 끊기면 0으로** → 매일 방문하게 만들어 광고 노출 누적

**조합 추천**: 희소성(#4), 매몰비용(#9)

---

## 4. 희소성 · FOMO (Scarcity, Fear of Missing Out)

- **자극**: "남들은 다 갖고 있는데 나만 없을까봐"
- **행동 유도**: 한정 수량·마감 임박·선착순·"3명이 보고 있습니다" 표시 → 충동 결제
- **수익 환금**:
  - 한정판 **프리미엄 가격** 수용도↑
  - 사전예약·대기열로 **수요 측정 + 현금 선입금**
  - 시즌·드랍 모델로 *재고 리스크↓ + 매출 피크 조절*

**조합 추천**: 사회적 증거(#5), 구별짓기(#6)

---

## 5. 사회적 증거 (Social Proof)

- **자극**: "남들이 하니까 안전하다/옳다"
- **행동 유도**: 베스트셀러 표시·후기 수·"OO명이 구매" 카운터 → 결정 마찰 제거
- **수익 환금**:
  - **전환율 직접 상승** (망설임 단축)
  - 초기에 **본인이 사주거나** 구매수를 만들어 첫 진입 장벽 돌파 (쇼핑몰 케이스)
  - 후기·평점 시스템 → *유저가 마케팅 콘텐츠를 무료로 생산*

**원본 케이스**: 코디 마켓에서 본인이 일반 고객인 척 구매해 첫 판매 흔적 만들기
**조합 추천**: FOMO(#4), 경쟁(#1)

---

## 6. 소속 · 구별짓기 (Belonging, Status)

- **자극**: "우리 편에 속하고 싶다" + "남들과는 다르고 싶다" (상반된 두 욕구가 동시에 작동)
- **행동 유도**: 커뮤니티 가입·VIP 등급·전용 컨텐츠 → 충성도·자존감
- **수익 환금**:
  - **멤버십·구독**(월 정기 결제)
  - VIP 전용 상품 **프리미엄 마진**
  - 커뮤니티 자체가 **락인** → 이탈률↓ → LTV↑

**조합 추천**: 경쟁(#1)으로 등급 가시화, 매몰비용(#9)으로 락인

---

## 7. 호혜성 (Reciprocity)

- **자극**: "받았으니 갚아야 한다"는 본능
- **행동 유도**: 무료 샘플·무료 컨설팅·선물 → 후속 구매율↑
- **수익 환금**:
  - **무료 체험 → 유료 전환** (가장 흔한 패턴)
  - 무료 콘텐츠로 신뢰 자산 축적 → 본 상품 가격 저항↓
  - "먼저 적립금을 드립니다" → 소진을 위한 재방문 유도

**조합 추천**: 손실 회피(#3) — 받은 걸 안 쓰면 손해

---

## 8. 자기효능감 (Self-efficacy, "내가 만들었다")

- **자극**: "내가 직접 했다는 성취감"
- **행동 유도**: DIY·커스터마이징·UGC → 결과물에 대한 애착과 자랑 욕구
- **수익 환금**:
  - **유저가 콘텐츠/상품을 직접 만들게** → 생산 비용↓ + 그 유저가 자기 작품을 자기 SNS에 자랑 (무료 마케팅)
  - 본인이 만든 작품이라 **반품률↓**, **재구매율↑**
  - "내 작품이 팔렸다" → 매몰비용 + 인정 욕구 동시 작동 → 재제작 사이클

**원본 케이스**: 코디 마켓 — 유저가 코디 만들어 팔리면 적립금 + 자기 블로그에 퍼감
**조합 추천**: 인정(#2), 매몰비용(#9)

---

## 9. 매몰비용 · 진행률 (Sunk Cost, Progress Bar)

- **자극**: "여기까지 쌓아온 게 아까워서 못 끊는다"
- **행동 유도**: 적립금·연속 출석·진행률 표시·레벨 → 중단 비용 인식
- **수익 환금**:
  - **이탈 방지** (LTV 직접 영향)
  - "거의 다 왔어요" 진행률 표시 → 결제·완주율↑
  - 적립금 시스템 자체가 **현금화 못 하는 화폐** → 자사몰 락인

**조합 추천**: 손실 회피(#3), 소속(#6)

---

## 10. 통제감 / 선택 환상 (Sense of Control)

- **자극**: "내가 선택했다"는 느낌
- **행동 유도**: 옵션·커스터마이징·"당신만의 OO" → 결정한 것에 책임감
- **수익 환금**:
  - 옵션 자체가 **추가 판매**(업셀)
  - 본인이 선택했기 때문에 **반품 저항** (자기 책임 인식)
  - A/B 선택 화면이 *데이터 수집 도구*가 되어 추천 정확도↑

---

## 11. 호기심 갭 (Curiosity Gap)

- **자극**: "다음에 뭐가 나오지?" — 변동 보상(variable reward)
- **행동 유도**: 랜덤 박스·가챠·"오늘의 추천"·다음 화 예고 → 반복 진입
- **수익 환금**:
  - **랜덤 박스** 직접 판매 (가장 강력한 단일 수익 모델)
  - 변동 보상으로 *체류 시간↑* → 광고·노출↑
  - "내일 다시 와야 결과를 본다" → DAU 유지

**경고**: 도박성에 가까울수록 규제·평판 리스크. 선제 방어 필수.

---

## 12. 우월감 / 구별짓기 (Distinction, Anti-mass)

- **자극**: "남들과는 다르다"
- **행동 유도**: 시리얼 넘버·한정판·"초대제" → 가격 무감각
- **수익 환금**:
  - **프리미엄 가격대** 자체가 성립
  - 초대제·웨이팅리스트로 *욕구를 미리 부풀린 뒤* 정가에 판매
  - "VIP 전용" 상품·서비스가 일반 등급의 *목표*가 되어 등급 상승 결제 유도

**조합 추천**: 희소성(#4), 소속(#6)

---

## 레버 조합 패턴 (잘 통하는 묶음)

| 조합 | 효과 | 예 |
|---|---|---|
| 경쟁 + 매몰비용 | 끊기 싫게 만든다 | 게임 시즌패스, 등급 시스템 |
| 인정 + 자기효능감 | 콘텐츠 생산자 만들기 | UGC 플랫폼, 코디 마켓 |
| 희소성 + 사회적 증거 | 충동 결제 극대화 | "10명이 보고 있고 재고 2개" |
| 호혜성 + 손실 회피 | 무료 → 유료 전환 | 적립금 만료 알림 |
| 호기심 + 변동 보상 | 중독적 재방문 | 랜덤 박스, 알고리즘 피드 |

---

## 사용 규칙

1. **답변에 레버를 명시하지 않으면 잘못한 것이다.** "이 안은 [경쟁] + [희소성]을 자극합니다"처럼 라벨링.
2. **레버를 당겼다고 끝이 아니다.** 그 다음 줄에 *수익 환금 경로*를 반드시 붙인다.
3. **윤리 검토 한 줄.** 손실 회피·호기심 갭은 강력하지만 *기만적이거나 도박적*으로 흐르면 평판·법적 리스크. "이 레버를 너무 세게 당기면 어떤 백래시가 올지" 한 줄 적는다 (선제 방어 = 원칙 #3).
