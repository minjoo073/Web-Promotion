# /강팀업데이트

CEO 가 "강팀에 올려", "강팀 업데이트", "이 패턴 강팀에 박아", "`/강팀업데이트`"
라고 하면 다음을 *조용히* 수행한다.

## 동작

### 1) 실행

```bash
if [[ -f scripts/promote-to-team.sh ]]; then
  bash scripts/promote-to-team.sh $ARGUMENTS
elif [[ -f ~/.claude/bin/promote-to-team.sh ]]; then
  bash ~/.claude/bin/promote-to-team.sh $ARGUMENTS
else
  echo "promote-to-team.sh 없음 — '강팀 불러와' 로 먼저 강팀 박아주세요."
  exit 1
fi
```

`$ARGUMENTS` 가 비어 있으면 가장 최근 회의 폴더 자동 선택.
경로를 직접 지정하려면: `/강팀업데이트 .ai-team/meetings/2026-05-28-주제`

### 2) 보고 (CEO 시점)

스크립트 마지막 줄이 PR URL 입니다. 그대로 CEO 에게 보고:

> 강팀 본체에 학습 PR 올렸습니다.
> URL: <PR_URL>
>
> PR 안 "재사용 가능 패턴" 섹션의 3개 체크박스에 답을 채우고 머지하시면,
> 다음 세션부터 모든 프로젝트에 그 학습이 자동 적용됩니다.

### 3) 안내 — CEO 가 PR 머지 후 할 일

머지 직후, 다른 PC·세션에서 다음 한 줄로 학습된 강팀을 받아옴:

```bash
curl -fsSL https://raw.githubusercontent.com/1215kkm/Ai_Team/main/scripts/pull-team.sh | bash
```

또는 이미 강팀이 깔린 세션에서: "강팀 업데이트 받아" / `/강팀불러오기`.

## 금기

- **자동 머지 금지** — 학습이 *진짜 재사용 가능*한지는 CEO 판단.
- **회의 요약 빈 채로 올리기 금지** — 본문에 SUMMARY 마커가 비어 있으면
  먼저 강팀장이 회의록을 마감해야 한다고 안내.
- **메인 클로드가 이 작업 직접 실행 금지** — `Bash` 툴로 위 스크립트만 호출.
  내용 *판단* (어느 패턴이 재사용 가능한지) 은 CEO 가 PR 페이지에서.
