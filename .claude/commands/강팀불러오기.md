# /강팀불러오기

사용자가 "강팀 불러와", "Ai_Team에서 강팀 가져와", "이 레포에 강팀 박아",
"`/강팀불러오기`" 라고 하면 *조용히* 아래를 수행한다.

`$ARGUMENTS` 가 비어 있으면 기본값 (`1215kkm/Ai_Team@main`, 현재 레포에 박기).
다른 포크/브랜치를 쓰려면 `$ARGUMENTS` 를 그대로 옵션으로 넘김.

---

## 수행

### 1) 사전 확인 (병렬)

```bash
test -f ~/.claude/agents/pm.md && echo "전역 있음" || echo "전역 없음"
test -d .claude/agents && echo "로컬 있음" || echo "로컬 없음"
git rev-parse --is-inside-work-tree 2>/dev/null && echo "git 레포 맞음"
```

이미 *전역 + 로컬 둘 다* 있으면 사용자에게 "이미 강팀이 깔려 있습니다 —
업데이트할까요?" 한 줄 물어본다. 답이 "응"이면 진행, 아니면 종료.

### 2) 본체 실행

로컬 스크립트가 있으면 그걸 쓰고, 없으면 원격 한 줄로 받음:

```bash
if [[ -f scripts/pull-team.sh ]]; then
  bash scripts/pull-team.sh $ARGUMENTS
elif [[ -f ~/.claude/bin/pull-team.sh ]]; then
  bash ~/.claude/bin/pull-team.sh $ARGUMENTS
else
  curl -fsSL https://raw.githubusercontent.com/1215kkm/Ai_Team/main/scripts/pull-team.sh \
    | bash -s -- $ARGUMENTS
fi
```

### 3) 보고 (한 줄)

설치된 멤버 수와 다음 명령만 보고한다 — 긴 설명 금지:

> 강팀 도착 (5명 + 텔레그램 발송 준비됨). `/회의시작` 으로 시작하세요.

텔레그램 설정이 없으면 한 줄 추가:
> 텔레그램 받으려면 한 번만: `bash ~/.claude/bin/setup-telegram.sh`

---

## 금기

- **회의 자동 시작 금지** — 강팀 설치만. 회의는 사용자가 `/회의시작` 으로 직접.
- **기존 `CLAUDE.md` / `.claude/settings*.json` 덮어쓰기 금지** — `pull-team.sh` 가 이미 보호하지만 확인 한 번 더.
- **긴 설명 금지** — 강팀은 도구다. 도구 박는 데 5줄 이상 쓰지 않는다.
